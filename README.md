## Install (macOS / Linux)

```bash
$ curl -sSL https://git.io/JUOVA | bash
```

after that, you will get `/usr/local/bin/gobackup` command.

[configure](./gobackup.yml):
```bazaar
$ vim ~/.gobackup/gobackup.yml
```

execute:
```bash
$ gobackup perform
```
